Who am I?<\/h1>\n

I am Thomas Langston, a professional software developer. I currently reside in New York City.<\/p>\n

In my programming career I’ve done feature work on B2B SaaS applications and a number of different developer tools for internal use. Other tools I’ve used and taught others to use at work to increase my team’s efficiency include Git, Bamboo, JIRA, Visual Studio, Selenium, and Resharper.<\/p>\n
